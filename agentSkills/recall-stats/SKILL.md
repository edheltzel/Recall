# source-command-recall-stats

Use this skill when the user asks to run the migrated source command `Recall-stats`.

## Command Template

Display database statistics for the Recall memory system, including record counts per table, database file size, and embedding status.

## Usage

```bash
recall stats
```

No arguments or options. Shows a complete overview of the database state.
